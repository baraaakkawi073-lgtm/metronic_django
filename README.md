# Horizon Solutions Dashboard

A modern **Enterprise Dashboard UI** built using **Django**, **Bootstrap**, and the **Metronic design style**.  
This dashboard provides an overview of business performance including users, visits, sales, revenue, and analytics charts.

## 📊 Dashboard Overview

The dashboard includes the following components:

### 1. Summary Cards
Quick statistics displayed at the top of the page:

- **Total Users** – Number of registered users
- **Total Visits** – Website visits for the current month
- **Total Sales** – Completed transactions
- **Revenue** – Total earnings

Each card shows growth or decline compared to last week.

---

### 2. Performance Analytics

A **line chart** showing monthly performance metrics:

- Revenue
- Users

This chart helps track growth trends throughout the year.

---

### 3. User Distribution

A **donut chart** displaying customer segmentation:

- Enterprise
- SMB
- Startup
- Personal

This allows quick visualization of the customer base distribution.

---
